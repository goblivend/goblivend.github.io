Ball Balancing Robot

A C++ robotics prototype that combines a camera-driven control loop, servo control, and balancing logic for a physical build.

A hardware-focused project organised into small modules: a camera input pipeline, a ball controller, PCA9685 servo driver bindings and SG90 wrappers. The repo includes 3D-print models and example control loops used to tune the balancing PID.

C++ CMake Embedded

Sudoku Solvers

A multi-language Sudoku solver set with brute-force and procedural strategies, split across C++, Java, and Rust implementations.

Implementations span Java, C++ and Rust to explore both algorithmic clarity and raw performance. The Java versions were used to prototype strategies and verify correctness, the C++ tree focuses on low-level optimisations and memory layout, and the Rust code includes Criterion benchmarks to compare solver variants under realistic inputs.

Across implementations the approach blends backtracking with procedural constraint propagation, reducing search space dramatically compared to naive brute-force. The repo is organised to let readers compare tradeoffs between clarity, speed and memory.

C++ Rust Java Benchmarks

Raytracer

A Java ray-tracing sandbox focused on reflections, spheres, planes, and experimenting with the math behind light interactions.

The code explores camera-based ray casting, intersection math and recursive reflections. Scenes include combined primitives (spheres, planes) and small scene-builder helpers for composing mirror-like arrangements; the README documents profiling notes used during experimentation.

Java Maven Rendering

The app uses OpenLayers with theme-aware basemap layers and a compact game flow for presenting rounds, scoring and revealing the true coordinates. Difficulty presets and guess-history overlays support learning and replayability.

tagForge

A browser-based local audio tag editor that uses the File System Access API to scan folders, edit metadata, rename files, and export playlists without a backend.

tagForge reads local file handles into an editable library, supports cover-art editing and filename token-based renaming. The implementation targets Chromium-based browsers due to File System Access APIs used for writes and renames.

tennis-generator

An interactive tennis-ball template generator built around a math-driven point generator and 4-way symmetry on a custom canvas grid.

The core algorithm is exposed in a single module (ball-generator.js) that projects user points through 4-way symmetry and auto-scales the canvas for 1:1 rendering. It’s a concise example of placing math at the centre of a small UI tool.

UI/UX created with AI; math & logic by author

JavaScript Canvas Math
